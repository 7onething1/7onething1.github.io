# Marvel Commander Night: Marketing Materials Kit

A ready-to-run marketing kit for hosting a **Doom Prevails** Marvel Super Heroes Commander night at your table or local game store. Copy-and-paste poster copy, table tents, social captions, a deck-tech link hub, and an organizer's run-of-show checklist — all in the Doom Prevails house style.

Live page: https://7onething1.github.io/wpn-marketing-materials/

## What's in the kit

- **Event poster copy** — a styled poster block plus a plain-text version for flyers and store event pages.
- **Table tents** — four fold-in-half cards: Doom's Table, Loki's Table, House Rules, and a New-here primer.
- **Social captions** — short, medium, and long lengths with ready hashtags.
- **Deck-tech link hub** — cards pointing to the Doom Prevails, Loki, rankings, and guides pages.
- **Run-of-show checklist** — the organizer's short list for a smooth night.

Each copy block on the page has a one-tap Copy button.
