    What is this? 🦸

    This is a scoreboard for a card game called Magic. Two players, Ben and Brandon, each start with a big number called life.

    The huge number in each box is how much life that player has. When it hits zero, they lose.

    Tap the green + buttons to add life. Tap the red − buttons to take life away. The 1 buttons change it a little, the 5 buttons change it a lot.

    The Commander damage boxes count hits from the other player's hero. If a hero deals 21, that player is knocked out and you see a big K.O.!

    Press New Game to start over. That's it. Have fun! 🎉
