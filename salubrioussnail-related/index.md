# Explore Related Commander Channels

## Similar channels putting out recent content

Channels and sites putting out recent Commander content in the same theory and deckbuilding lane as Salubrious Snail. Salubrious Snail sits at the essay and principle end, so the closest matches are data and philosophy shows rather than pure gameplay.

- **EDHRECast** (Podcast and data): The official EDHREC show. Weekly episodes on deckbuilding data, card evaluation, and format trends. The closest active match to Snail's data driven theory videos.
- **The Command Zone** (Strategy and social): Josh Lee Kwai and crew cover threat assessment, politics, and deck strengths and weaknesses. Overlaps Snail on strategy theory, lighter on hard deckbuilding math.
- **EDH Deckbuilding** (Deckbuilding how to): A channel built entirely around the deckbuilding process itself, which is Snail's core subject. Frequent recent uploads.
- **Commander's Herald** (Articles): Written theory and philosophy pieces on power level, rule zero, and archetype design. Text companion to Snail's video essays.
- **EDHREC Articles** (Articles): Philosophy Friday and Commander Mechanic run recurring deckbuilding principle columns that map almost one to one onto Snail's topics.

## Command Power: a cross-channel index

We already hold the full Command Power catalog on disk (526 videos). Command Power is a card review and deck tech channel, so its overlap with Snail's theory essays is narrow. These are the Command Power videos we have that sit in the same theory and deckbuilding lane as Snail, rather than single card reviews.

- **Why I Have (Mostly) Stopped Using EDHREC** vs Snail *How EDHRec Data Aggregation Warps Commander Deckbuilding*: Direct theme match. Both argue that leaning on EDHREC aggregate data flattens deckbuilding.
- **Budget Deck Tech series (Feldon, Neyam Shai Murad, Calamity, and more)** vs Snail *Building a Powerful Budget Malcolm Francisco Deck*: Shared budget building lane, though Command Power does it deck by deck while Snail teaches the budget principle.
- **Precon Upgrade series (Peace Offering, Death Toll, Endless Punishment)** vs Snail *Common Commander Deckbuilding Mistakes and Solutions*: Adjacent, not identical. Upgrades apply fixes that Snail frames as general mistakes to avoid.

Command Power and Salubrious Snail are different genres. The honest overlap is small and centered on EDHREC skepticism and budget building. Use Command Power for card and deck specifics, Snail for the underlying theory.
