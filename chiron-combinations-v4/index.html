This gathers, in one place: an automated navigation summary, Chiron's own meaning for each card, and her recorded evidence for each pair, grouped by evidence type and quoted verbatim with the source video. One caveat the data forces: the corpus records which cards appear together, not which came first, so a pair reads the same in either direction. This is co-occurrence, not ordered syntax. Methodology, sources, and what this does not cover

Source

Corpus collected from 446 Chiron Intuition transcripts through July 7, 2026. Every quoted line is verbatim and links to its source video. Links are video-level, not timestamped: the corpus stores the video id but not the moment, question, surrounding cards, or original card order, so those are not shown. A grounding pass confirmed each read traces to its cited video.

What this page mixes (three sources, labeled)

The page is not a single voice. It carries three kinds of text, kept distinct:
  • Verbatim Chiron quotations — the pair passages and the raw card mentions, each source-linked.
  • Chiron's curated meaning — the keyword line and one or two sentences per card, from her own cheatsheet, plus a clarifier note when one names the other card.
  • Automated summaries — the blocks marked "Automated corpus summary" are generated from keyword tags, not quoted from Chiron. Each states the rule that produced it.
A fourth, outside layer is the Bunning structural label. It is not Chiron's, so it sits behind an "outside comparison" toggle under each pair's evidence, never ahead of it.

Ordered evidence, not yet supported

The current corpus is unordered co-occurrence. Forward and reverse of a pair return the same passages, so the page does not claim a first-position card reads differently from a second-position one. Genuine ordered syntax would require rebuilding the corpus from the transcripts with each card's position and timestamp retained. That rebuild is queued.

Coverage

This build indexes 2822 of the 3003 possible unordered pairs (94%) across the 78 cards, from 26,169 combination passages, about 9.3 per pair. A pair she was never recorded reading shows “No recorded Chiron pairing found” rather than an invented reading.

Preservation audit

An acceptance test (audit.py, shipped beside this page) checks that every corpus record reaches a rendered result. Result: 26,169 combination passages (direct 1,892; sequence 7,334; clarifier 5,966; nearby 10,977), of which 16,991 are clean and 9,178 are flagged (fragment, negative, third-card, or near-duplicate). Plus 3,523 standalone card mentions. Every passage is reachable through the pair's grouped "view all" list, every mention through each card's raw toggle, and 181 pairs are confirmed gaps with no passage. Unreachable records: 0.

The structural lens (Bunning)

Each pair is classified against Joan Bunning's Learning the Tarot. Two labels come from a verified relationship table (bunning_relations.json) extracted from her per-card Reinforcing and Opposing Cards pages plus her Lesson 15 named pairs; two are computed from card identity, which Bunning treats as reliable; the rest default to Occasional:
  • Opposing pair — verified: both cards appear on each other's Opposing Cards list in Bunning's book (for example the Magician and the High Priestess).
  • Reinforcing pair — verified: both cards appear on each other's Reinforcing Cards list (for example the Empress and Nine of Cups). Same suit alone never qualifies.
  • Ace pair — computed: both cards are Aces.
  • Court pair — computed: both cards are court cards.
  • Occasional pair — default: no verified relationship and not a court or Ace pair. Any two cards can still be read together for one reading.
Coverage note: the verified table currently covers the Minor Arcana card pages plus Lesson 15's named Major pairs. A pair with no verified entry falls to Occasional rather than being guessed. Completing the Major Arcana extraction is queued.
